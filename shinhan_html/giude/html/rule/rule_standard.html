입력, 선택 컨트롤 input, textarea, select
  • type에 맞는 입력필드를 사용한다. (tel, text, email, search, password)
  • 숫자전용필드는 "tel"을 사용한다.
  • IOS : 입력필드는 기본값으로 라운드를 초기화 한다.
  • IOS : 줄간격은 폰트사이즈+10 으로 한다. (textarea는 시안우선)
  • MAC : 검색필드는 CSS 호환이 안되므로 지양한다.
