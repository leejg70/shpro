Publishing Conventions

공백

  • 들여쓰기는 탭 으로 설정한다. (공백 4칸 사이즈)
  • 빈줄은 1줄을 초과하지 않는다.

주석

HTML

								[기본 주석]
								
								

								[조건부 주석]
								
								

								[변경사항 주석]

CSS

								[파일 주석]
								/*-------------------------------------------------------------------
									파일정의 :
									...
									(생략)
								-------------------------------------------------------------------*/

								[분류 주석]
								/*-------------------------------------------------------------------
									@Fonts
								-------------------------------------------------------------------*/

								[구분 주석]
								/* Default */

SCRIPT

								[파일 주석]
								/*-------------------------------------------------------------------
									분류순서
									...
									(생략)
								-------------------------------------------------------------------*/

								[분류 주석]
								/*-------------------------------------------------------------------
									@Variables
								-------------------------------------------------------------------*/

								[구분 주석]
								/* 엘리먼트 설정 */

								[설명 주석]
								// 스와이프 적용클래스
  • 목적에 맞게 주석표기 방법을 구분한다.
  • 범위에 맞게 주석표기 크기를 구분한다.
  • 분류순서와 분류별 주석표기 '@'로 찾기 쉬워야 한다.

순서

  • 분류의 우선순위대로 적용한다.

HTML

HTML 속성 선언순서
분류 구분 속성
1. 필수 기능 href, src, type
2. 일반 이름 name, id
정보 value, title, alt
옵션 target, tabindex
3. 표현 스타일 class, style
4. 기타 role role
aria- aria-controls, aria-describedby, aria-labelledby, aria-popup, aria-modal 등
data- data-id, data-target 등
상태 checked, disabled, selected, required, hidden 등

CSS

CSS 속성 선언순서
분류 의미 대표되는 속성(그룹) 관련 속성
1. 레이아웃 가상선택자 contents:'';
흐름 overflow
위치 position, top, right, bottom, left
넘침 float, clear left, right, both
표시 visibility, display visible, hidden, block, inline-block, inline, flex, inline-flex, table, table-cell
2. 크기/간격 크기 width, *-width, height, *-height
간격 margin, padding
3. 폰트/정렬 폰트 font-* (body와 동일한 폰트사이즈는 생략한다.) font-weight, font-size, font-family
컬러 color
정렬/꾸밈 등 line-height, letter-spacing, text-align, vertical-align, text-decoration, text-indent, white-space, cursor
4. 박스모양 선 border, border-radius border-width, border-style, border-color
배경 background background-color, background-image, background-repeat, background-position, background-size
박스 box-sizing, box-shadow
5. 동작 동작 animation, transform, transition
6. 배치 배치 z-index
7. 기타 언급되지 않은 속성들은 후순위로 적용한다.

작성법

  • 타입과 속성 사이는 스페이스 한 칸으로 적용한다.
  • 세미콜론 뒤 다음 타입은 스페이스 없이 적용한다.
